# Telegram Channel Widget

Виджет для сайта, который показывает:
- канал Telegram и его название;
- количество подписчиков;
- 4 последних поста;
- лайки (реакции), комментарии и просмотры по постам;
- кнопку "Открыть канал".

## Канал по умолчанию

`r77_ai`

Изменить можно в файле `public/widget.js`:

```js
const CHANNEL = "r77_ai";
```

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить сервер:

```bash
npm start
```

3. Открыть в браузере:

`http://localhost:3000`

## Важно

- Данные берутся с публичной страницы `https://t.me/s/<channel>`.
- Если Telegram изменит HTML-структуру страницы, парсер может потребовать обновления.
